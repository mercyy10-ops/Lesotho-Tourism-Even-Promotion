# Lesotho-Tourism-Even-Promotion
This 5-page website for the Maletsunyane Falls Festival in Lesotho uses HTML and CSS to attract global tourists (p. 1). It features an immersive Home Page with an event banner, an "About" section detailing the festival's heritage, a scheduled activity table, a secure registration form, and a contact page with social media and map links (p. 2).
